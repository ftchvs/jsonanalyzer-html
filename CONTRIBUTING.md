# Contributing

Thanks for taking a look at JSON Analyzer HTML.

## Local Development

This project has no install step. Open `index.html` directly or run:

```sh
python3 -m http.server 8080
```

## Before Opening a Pull Request

- Test the sample data flow with **Load sample data**.
- Test uploading at least two local `.json` files.
- Confirm CSV and JSON exports still download.
- Include a screenshot for UI changes.
- Do not commit private JSON payloads, credentials, logs, or customer data.
